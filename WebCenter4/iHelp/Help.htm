WebCenter help
WebCenter is a browser based approval client for controlling page content in ControlCenter Enterprise.
The page previews and thumbnails are generated by the ControlCenter InputCenter application and made accessible for the web server.
All browser users are required to log in using username and password.
There are three main windows (views) in Webcenter, described on the following pages

Tree view (grouping of products for page filtering)
Pages view (page gallery)
Zoom view (the actual page inspection window)
